## 🚀 About Me

Full-stack AI engineer and indie maker from **India** — I build products that make AI feel human. Currently **Director of Engineering at [Hyperbond Studio](https://hyperbond.ai)** (Call Me Sensei), and shipping my own AI tools on the side.

- 🧠 **AI-first**: voice, agentic workflows, real-time phone AI, emotional engagement systems
- 💼 **$300K+ earned** on Upwork · ⭐⭐⭐⭐⭐ Top Rated · 14+ years building
- 🎙️ Built **DictationDoggy** — macOS voice dictation powered by Whisper + LLM intent routing
- 📱 Published **2 native Expo modules** on npm (AAC→M4A, multilingual speech)
- 🌍 Open-source: **247-country visa dataset** used as a public API by travel apps
- ⚡ I move fast: idea → MVP → shipped, often in days
